Installation:
---------------
Installation file name is  "xfee.ocmod.zip".  It is available inside respective directory for your opencart version.

1. Please Go to Extensions -> Extension Installer
2. Upload "xfee.ocmod.zip" and click on continue.
3. Now install from admin -> Extensions -> Filter by Order Totals -> X-Fee
